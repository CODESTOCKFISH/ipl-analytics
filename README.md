# 🏏 IPL Analytics Dashboard

An end-to-end data analytics project analyzing IPL performance using custom metrics and interactive visualizations.

---

## 🚀 Features

### 📊 Phase-wise Analysis
- Powerplay (Overs 1–6)
- Middle Overs (7–15)
- Death Overs (16–20)
- Overall performance

### 🏏 Batting Insights
- Most runs
- Strike rate (volume-adjusted using SR × log(balls faced))
- Phase-based performance comparison

### 🎯 Bowling Insights
- Economy rate
- Strike rate
- Wickets per over
- Custom impact score balancing efficiency + wicket-taking ability

### 🔥 All-Rounder Rankings
- Combined batting + bowling performance
- Includes **role-based contributions (even part-time players depending on phase)**

### 🔍 Search Functionality
- Easily search any player across datasets

---

## 📈 Key Highlights

- Built **custom impact metrics** instead of relying on raw stats  
- Solved real-world issues like:
  - Small sample size bias
  - Misleading strike rates
  - Low-volume players distorting rankings  
- Phase-wise segmentation to reflect **actual match situations**
- Clean and interactive UI using **Streamlit**

---
